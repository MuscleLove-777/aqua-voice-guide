Aqua Voiceのインストールと初期設定

Aqua Voiceを使い始めるのは非常に簡単です。以下のステップで、あなたの開発環境にAqua Voiceを導入しましょう。

  1. Aqua Voiceのダウンロード: まず、Aqua Voiceの公式サイト（https://withaqua.com）にアクセスし、お使いのOS（Windows/macOS/Linux）に合わせた最新版インストーラーをダウンロードします。ダウンロードボタンはページ上部に分かりやすく配置されています。
  2. インストーラーの実行: ダウンロードしたファイルをダブルクリックしてインストーラーを起動します。画面の指示に従い、「次へ」をクリックしてインストールを進めます。インストール先ディレクトリはデフォルト設定で問題ありませんが、必要に応じて変更することも可能です。
  3. アカウント作成とログイン: インストール完了後、Aqua Voiceが自動的に起動します。初回起動時には、アカウント作成またはログインを求められます。既存のGitHub/Googleアカウントとの連携も可能で、迅速にセットアップを完了できます。
  4. マイク設定とキャリブレーション: Aqua Voiceは初回起動時にマイクの自動検出とキャリブレーションを行います。指示に従って、マイクに向かっていくつか文章を話すことで、あなたの声質や環境ノイズに最適な設定が自動的に適用されます。これにより、音声認識精度が大幅に向上します。高品質なヘッドセットマイクの使用を強く推奨します。
  5. 基本設定の最適化: 設定メニューから、起動時の自動実行、音声フィードバックのオン/オフ、認識言語（日本語、英語など複数言語に対応）を選択できます。また、未来予測型のAIアシスタント機能（{{internal_link:Aqua Voiceの最新アップデート情報}}で詳細をチェック）もここで有効化・調整可能です。

基本的な音声コマンドと操作

Aqua Voiceは自然言語に近い感覚でコードを入力・操作できます。よく使う基本的なコマンドを習得しましょう。

  • テキスト入力: 単語やフレーズを話すだけで入力されます。「ハロー ワールド」と話せば Hello World が入力されます。
  • 特殊記号の入力: プログラミングに不可欠な記号も音声で簡単に入力できます。
    • 「セミコロン」: ;
    • 「コロン」: :
    • 「ドット」: .
    • 「カンマ」: ,
    • 「スラッシュ」: /
    • 「バックスラッシュ」: \
    • 「アンダースコア」: _
    • 「ダッシュ」または「ハイフン」: -
    • 「アットマーク」: @
    • 「ハッシュ」または「シャープ」: #
    • 「ドル」: $
    • 「パーセント」: %
    • 「アンド」: &
    • 「アスタリスク」: *
    • 「パイプ」: |
  • 括弧類の入力: ペアで入力されるため、閉じ括弧を忘れる心配がありません。
    • 「丸括弧」: ()
    • 「角括弧」: []
    • 「波括弧」: {}
    • 「山括弧」: <>
  • カーソル移動と選択: 音声で自由にカーソルを操作し、テキストを選択できます。
    • 「左へ N 文字」: カーソルを左へN文字移動
    • 「右へ N 文字」: カーソルを右へN文字移動
    • 「上へ N 行」: カーソルを上へN行移動
    • 「下へ N 行」: カーソルを下へN行移動
    • 「行頭へ」: 行の先頭へ移動
    • 「行末へ」: 行の末尾へ移動
    • 「単語選択」: 現在の単語を選択
    • 「行選択」: 現在の行を選択
  • 編集操作: 基本的な編集コマンドも音声で実行できます。
    • 「削除」: 現在のカーソル位置から一文字削除
    • 「N 文字削除」: N文字削除
    • 「アンドゥ」: 直前の操作を取り消し
    • 「リドゥ」: 取り消した操作をやり直し
    • 「コピー」: 選択範囲をコピー
    • 「ペースト」: クリップボードの内容を貼り付け
    • 「カット」: 選択範囲を切り取り

カスタム辞書で認識精度を向上させる

Aqua Voiceの真価は、その高度なカスタム辞書機能にあります。プログラミングには一般的な単語とは異なる独自の用語や略語が頻繁に使われるため、この機能は開発者にとって不可欠です。

  1. カスタム辞書の作成: Aqua Voiceの設定画面を開き、「カスタム辞書」タブを選択します。「新しい辞書を作成」をクリックし、辞書名を入力します。
  2. 用語の追加: 新しい辞書に、認識させたい単語とその読み方を登録します。
    • 例1: 会社やプロジェクト固有の略語
      • 読み: 「アクア ボイス」 → 語彙: AquaVoice
      • 読み: 「シーエムエス」 → 語彙: CMS (Content Management System)
    • 例2: 一般的な単語だが、特定の表記をさせたい場合
      • 読み: 「ユーザー インターフェース」 → 語彙: UserInterface (パスカルケース)
      • 読み: 「レガシー コード」 → 語彙: legacyCode (キャメルケース)
    • 例3: 発音しにくい、または誤認識されやすい技術用語
      • 読み: 「イテレーター」 → 語彙: Iterator
      • 読み: 「シンギュラリティ」 → 語彙: Singularity
  3. 複数パターンの登録: 一つの語彙に対して複数の読み方を登録することも可能です。例えば、「ギット」と話しても「ジット」と話しても Git と認識させたい場合、両方を登録します。
  4. 辞書のインポート/エクスポート: 大規模なプロジェクトやチームで作業する場合、共通のカスタム辞書をXMLやJSON形式でインポート・エクスポートし、共有できます。これにより、チーム全体の認識精度を均一に保ち、初期設定の手間を省くことができます。

カスタム辞書を積極的に活用することで、 Aqua Voiceはあなたの話し方に合わせて進化し、より正確で効率的な音声コーディング環境を実現します。

開発者向け活用テクニック

Aqua Voiceは、主要な開発環境とシームレスに連携し、開発ワークフロー全体を音声でコントロールする強力な機能を提供します。

VSCodeでの音声コーディング

Visual Studio Codeは、その拡張性の高さから多くの開発者に愛用されています。Aqua VoiceはVSCode専用の公式拡張機能（2026年時点）を通じて、以下のような高度な連携を実現しています。

  • コードスニペットの音声呼び出し: 「スニペット クラス 定義」と話すだけで、あらかじめ定義しておいたクラスの雛形が展開されます。例えば、Reactのコンポーネント定義を「コンポーネント React」で呼び出す、といった活用が可能です。
  • コマンドパレットの操作: 「コマンド パレット」と話してパレットを開き、「ファイル 保存」や「フォーマット ドキュメント」といったVSCodeの組み込みコマンドを音声で実行できます。
  • デバッグ操作: 「デバッグ 開始」、「ステップ オーバー」、「ブレークポイント トグル」など、デバッグプロセスも音声で制御できます。これにより、両手がキーボードから離れている状態でも、スムーズにデバッグ作業を進められます。
  • ファイル・ディレクトリ操作: 「ファイル 開く」、 「ファイル 新規」、「フォルダ 作成」など、プロジェクト内のファイルやディレクトリを音声で操作できます。

JetBrains IDEでの活用

IntelliJ IDEA、PyCharm、WebStormなどのJetBrains製IDEも、Aqua Voiceの深い統合の恩恵を受けられます。組み込みのVoice Assistant機能（2026年時点）と連携し、より自然な音声操作が可能です。

  • ナビゲーション: 「ファイル ナビゲート」、「クラス 検索 メインアクティビティ」、「メソッド 定義 ユーザーログイン」など、広大なコードベース内を素早く移動できます。
  • リファクタリング: 「リファクタリング メソッド 抽出」、「変数 名 変更 ユーザーデータ」といった高度なリファクタリング操作を音声で指示できます。これにより、コードの品質を保ちながら、スムーズな改善が可能です。
  • ライブテンプレート: JetBrains IDEの強力なライブテンプレート機能も音声で起動できます。「テンプレート フォー ループ」と話すだけで、for ループの基本構造が生成され、その後、変数名などを音声で補完していきます。

ターミナル操作を音声で

コマンドラインインターフェース（CLI）での作業は、開発者にとって日常の一部です。Aqua Voiceはターミナル操作も完全にサポートし、効率を向上させます。

  • 基本コマンドの実行: 「エルエス」、「シーディー ドキュメント」、「メイクディア プロジェクト名」、「ギット ステータス」といったコマンドを音声で実行できます。
  • 複雑なコマンドの入力: パイプ（|）やリダイレクト（>）を含む複雑なコマンドも、「コマンド グレップ パイプ ソート」のように、区切りを意識して話すことで正確に入力されます。
  • 環境変数の設定: 「エクスポート パス ユーザー ローカル ビン」など、環境変数の設定も音声で行えます。

Git操作の音声コマンド

バージョン管理システムであるGitも、Aqua Voiceを使えばハンズフリーで操作可能です。

  • コミット: 「ギット コミット メッセージ フィックス バグ」と話すだけで、コミットメッセージ付きでコミットを実行します。
  • ステージング: 「ギット アド ドット」で全ての変更をステージング。「ギット リセット ヘッド ファイル名」でステージング解除も可能です。
  • プッシュ・プル: 「ギット プッシュ オリジン メイン」、「ギット プル」など、リモートリポジトリとの同期も音声で指示できます。
  • ブランチ操作: 「ギット ブランチ チェックアウト フィーチャー 新機能」のように、ブランチの切り替えや作成もスムーズに行えます。
